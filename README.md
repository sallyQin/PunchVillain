# 打小人PunchVillain

## 分二大部分：
   1.打小人（又名扎小人）
   2.求解符
      2.1 算卦抽符
      2.2 九宫格抽财神签
      
###* 实现几个主要功能：

#####1.系统可通过相机、照片库以及系统图选小人。

#####2.自定义彩色进度条的设置。

#####3.开打前各类道具的选择和使用（带音效）。

#####4.模拟 算卦抽签过程，并随机获得签符（带音效）。

#####5.模拟 附带走马灯效果的九宫格抽财神签（带音效）。
